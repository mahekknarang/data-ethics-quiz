Participation Terms

Data Ethics live demo · Session-scoped · Honest disclosure

By joining this quiz, you agree to participate in a live classroom demonstration about data collection practices. This page is intentionally plain — the same way real product terms usually are.

What we collect

  • The display name you enter
  • Device information: browser, operating system, device type, screen size
  • Locale signals: language and timezone
  • Network metadata: IP address (stored server-side; not shown publicly)
  • Timing data: when you loaded the page, when you joined, time spent on the join screen
  • Quiz behavior: answers, correctness, and how quickly you answered each question
  • Raw user-agent string (for parsing device details)

Why

This data powers a live demonstration of how ordinary web apps silently assemble a behavioral and device profile from a normal signup flow — even when you only typed a first name. It is collected for educational purposes during this presentation.

How long we keep it

Session data is intended to be deleted after the presentation. It is not sold, shared with advertisers, or used for grading.

What we won't do on the reveal screen

The public dashboard shows aggregate patterns (device mix, average timings, anonymized hall-of-fame first names for fastest joiners). It does not publicly display IP addresses or full device fingerprints of individuals.

If you prefer not to participate, close this tab and don't join. No hard feelings.
